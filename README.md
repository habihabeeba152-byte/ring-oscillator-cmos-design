📌 Aim

To design and simulate a ring oscillator using CMOS NAND gates and inverters, and analyze the variation of oscillation frequency with respect to the number of stages.

📖 Introduction

A ring oscillator is a simple electronic oscillator consisting of an odd number of inverting stages connected in a feedback loop. Due to the propagation delay of each stage, the circuit produces a periodic oscillating output.

⚙️ Working Principle

The output of the last stage is fed back to the input of the first stage.
An odd number of inversions ensures oscillation.
Oscillation frequency depends on propagation delay.
🧮 Frequency Equation

f = 1 / (2 × N × tp)

Where:

N = Number of stages
tp = Propagation delay of each stage
